# Curriculum Mode

Es wurde kein Lernmaterial hochgeladen. Das Quiz entsteht aus Lehrplaninhalt, learning_goal, Unterthema und Lehrerwunsch.

- Nutze learning_goal aus curriculum_topic_topics und curriculum_topic_subtopics als verbindliche didaktische Grundlage.
- Erzeuge keine Materialanalyse und keine Formulierungen wie „im Arbeitsblatt“, „in der Quelle“ oder „laut Material“.
- Keine generischen Wikipedia-Fragen; jede Frage soll auf Lernziel, Kompetenz oder Unterrichtsthema einzahlen.
- Berücksichtige den benutzerdefinierten Prompt des Lehrers als Schwerpunktsetzung.
- Plane eine sanfte Schwierigkeitsprogression.
