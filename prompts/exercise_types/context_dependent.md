# Context Dependent Exercise Rules

- Wenn eine Aufgabe nur mit Bild, Tabelle, Wortbank, Übersetzungsspalte oder Begleittext lösbar ist, muss dieser Kontext im Fragetext sichtbar sein.
- Keine Verweise auf nicht sichtbares Material.
- Wenn Kontext nicht zuverlässig rekonstruierbar ist, erzeuge eine ähnliche neue Aufgabe zum gleichen Lernziel.
