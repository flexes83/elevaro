# Elevaro

**Elevaro – Spielerisch zu guten Noten.**

Elevaro ist eine spielerische Lernplattform für schulnahes Quizzen.
Der Fokus liegt auf konkretem Schulstoff nach Fach, Klasse und Thema.

## Aktueller Stand

Diese Version enthält:
- Startseite mit Produkttexten
- erstes spielbares Mathe-Quiz
- direkte Rückmeldung nach jeder Frage
- Ergebnisanzeige
- Wackelkandidaten-Modus
- lokale Fortschrittsspeicherung

## Technik

- PHP 8+
- Bootstrap 5
- Vanilla JavaScript
- JSON-basierte Quizdaten
